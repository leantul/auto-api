# Eliminar una unidad

Elimina la unidad 

**URL** : `/unit/{unitId}`

**Parámetros** : `{unitId}` es el id de la unidad a eliminar.

**Método** : `DELETE`

**Ejemplo:**
```json
{
  "additionalList": [
    {
      "additionalID": 0,
      "name": "string",
      "price": 0
    }
  ],
  "auto": {
    "autoId": 0,
    "name": "string",
    "price": 0
  },
  "price": 0,
  "unitID": 0
}
