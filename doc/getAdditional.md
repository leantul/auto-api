# Obtener adicional
Obtiene el adicional con el id enviado.

**URL** : `/additional/{additionalId}`

**Parámetros** : `{additionalId}` es el id del adicional a obtener. 

**Método** : `GET`

**Ejemplo: `/additional/2`** 
```json
{
  "additionalID": 2,
  "name": "Aire Acondicionado",
  "price": 20000.00
}
