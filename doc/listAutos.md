# Listar autos
Muestra todas las variantes existentes, junto con su precio.

**URL** : `/auto`

**Parámetros** : `Ninguno`

**Método** : `GET`

**Ejemplo:**
```json
[
  {
    "name": "Sedán",
    "price": 230000.00,
    "autoId": 1
  },
  {
    "name": "Familiar",
    "price": 245000.00,
    "autoId": 2
  },
  {
    "name": "Coupé",
    "price": 270000.00,
    "autoId": 3
  }
]
